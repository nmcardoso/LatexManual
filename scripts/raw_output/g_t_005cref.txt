            Synopsis:   \ref{key}   Produces the number of the sectional unit, equation, footnote, figure, , of the corresponding \label command (see \label).  It does not produce any text, such as the word Section or Figure, just the bare number itself.  In this example, the \ref{popular} produces 2.  Note that it is a forward reference since it comes before \label{popular}.   The most widely-used format is item number~\ref{popular}. \begin{enumerate} \item Plain \TeX \item \label{popular} \LaTeX \item Con\TeX t \end{enumerate}
